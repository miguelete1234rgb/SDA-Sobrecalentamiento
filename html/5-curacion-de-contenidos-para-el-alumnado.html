SDA SOBRECALENTAMIENTO MOTOR

5. Curación de contenidos para el alumnado

Recursos

Esta recopilación reúne recursos pensados para que el alumnado pueda entender mejor el sistema de refrigeración y elaborar el producto final.

Incluye recursos técnicos básicos sobre la estructura y función del sistema, funcionamiento de los componentes principales, averías habituales y tutoriales sencillos para usar Presentaciones de Google, Google Docs o Canva.

La finalidad de esta curación es que el alumnado no solo encuentre información, sino que tenga apoyo para transformarla en un producto claro, breve y ordenado.
